## Clone & Reproduce

1. Start by cloning this repo.
    ```javascript
    git clone https://github.com/stgonzales/github-explorer
    ```
2. Execute `yarn` to install all the packges.

3. And last but least, execute
    ```javascript
    yarn start
    ```
